##  Included Tools

Quick links to the following converters:
- **CSV to Excel**: Convert CSV files to Microsoft Excel format.
- **Excel to CSV**: Convert Excel spreadsheets to CSV.
- **JSON to CSV**: Transform JSON data into CSV format.
- **CSV to JSON**: Convert CSV data to JSON objects.
- **CSV to PDF**: Generate PDF documents from CSV files.
- **CSV to SQL**: Create SQL INSERT statements from CSV data.

### Manual Installation (Developer Mode)
1. Clone or download this repository.
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** in the top right corner.
4. Click **Load unpacked**.
5. Select the `chrome-extension` folder from this repository.

##  About CSVtoAny

[CSVtoAny.com](https://csvtoany.com) is a comprehensive online platform for data conversion. We provide free, secure, and fast tools to help developers, data analysts, and everyday users transform data between various formats including CSV, Excel, JSON, XML, SQL, PDF, and more.

- **Secure**: All conversions happen in your browser (for supported tools) or via secure processing.
- **Free**: No hidden costs or subscriptions.
- **Versatile**: Support for a wide range of file formats.
